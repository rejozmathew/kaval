# Kaval bootstrap pack

This folder contains the repo-control files for running Kaval with Codex + TaskMaster.

## Included
- `AGENTS.md`
- `STATUS.md`
- `plans/phase-0.md`
- `plans/phase-1.md`
- `plans/phase-2a.md`
- `plans/phase-2b.md`
- `plans/phase-3.md`
- `plans/phase-4.md`
- `docs/CODEX_TASKMASTER_SETUP.md`
- `.codex/config.example.toml`
- `.taskmaster/docs/prd.txt`
- `docs/prd.md`

## Intended use
1. Copy these files into the root of the new `kaval` repo.
2. Open the repo in VS Code using a WSL workspace.
3. Install and sign in to Codex.
4. Initialize TaskMaster and parse the PRD.
5. Start with Phase 0.
