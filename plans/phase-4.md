# Phase 4 — OSS Distribution

## Objective
Make Kaval installable and useful for other Unraid users within minutes.

## Deliverables
- Unraid Community Apps template
- Complete install documentation
- Contributor guides for descriptors and normalizers
- More descriptors
- Security audit
- Performance optimization
- README with screenshots/demo/feature list

## Exit criteria
- New user installs and sees service map within 2 minutes after API key entry
- Contributor can add descriptor without touching Python code
- Shipped descriptors pass contract tests
- No secrets appear in logs, prompts, or UI

## Notes
- Keep docs and screenshots aligned with the actual product state.
- Preserve ADR/PRD/CHANGELOG governance model.
